# demoModelDeployment
Ejemplos de cómo se puede realizar el deployment de modelo TensorFlow+Keras para clasificar Flores IRIS.

## Instalación
1. Acceder a [Google Drive](https://drive.google.com/drive/my-drive) usando una cuenta disponible.
2. Si no existe, crear la carpeta "**demosColab**" en la raíz de la unidad. 
3. Crear la carpeta "**demoModelDeployment**" dentro de la carpeta "demosColab" (ya existente o creada en el paso anterior).
4. Subir todos los archivos de este repositorio dentro de la carpeta creada en el paso anterior. 

## Uso
Seguir los pasos indicados en el archivo [**0-Instructivo.md**](https://github.com/PGP-MachineLearning/demoModelDeployment/blob/main/0-Instructivo.md).
